1. Open your code in VS code 

2. Start the Apache Server from the XAMPP control panel

3. Again start the MySQL server .

4. open your project in command prompt and then type 
     " php -S localhost:8000  "
    ( you can give any port no. just remind that no other
      system software using that port )

5. Open the browser and then type " http://localhost:8080/dashboard/ "

6. Select phpMyAdmin or type " http://localhost:8080/phpmyadmin/ "

7. For the Admin pannel dashboard  type " http://localhost:8000/admin/index.php " 

8 git add . ( to add all file).
